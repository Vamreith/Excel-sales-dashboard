# Dashboard sprzedaży (Excel)

## Cel projektu
Analiza sprzedaży produktów oraz prezentacja kluczowych wskaźników KPI.

## Wykorzystane narzędzia
- Microsoft Excel
- Tabele Excela
- Formuły (SUMA, ŚREDNIA, JEŻELI, LICZ.JEŻELI, SUMA.JEŻELI)
- Odwołania strukturalne
- Tabele przestawne
- Wykres przestawny
- Dashboard KPI
